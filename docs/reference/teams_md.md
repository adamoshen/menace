# Markdown formatting for Microsoft Teams

Format a character vector for Microsoft Teams.

## Usage

``` r
teams_bold(string)

teams_italic(string)

teams_strikethrough(string)
```

## Arguments

- string:

  A character vector.

## Details

Microsoft Teams uses a different Markdown syntax. See
[here](https://support.microsoft.com/en-us/office/use-markdown-formatting-in-microsoft-teams-4d10bd65-55e2-4b2d-a1f3-2bebdcd2c772)
for a full list of formats.
